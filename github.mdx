
컴퓨터 처음 세팅할때 ssh-keygen 등록(맨날 까먹는!! ㅋㅋ)
- 회사 컴에서는 토큰을 자주 삭제 후 재생성 + 등록할 예정
```bash
$ ssh-keygen
Generating public/private ed25519 key pair.
Enter file in which to save the key (/c/Users/tw-ms/.ssh/id_ed25519): /c/0.workspace/[경로생략]/chagchagchag/id_rsa
...

$ cd /c/0.workspace/[경로생략]/chagchagchag
...

$ ls
id_rsa  id_rsa.pub
```
